## 后端完成情况

- 建立 Java 21 + Spring Boot 3.x 后端工程。
- 建立模块化单体包结构：`common`、`config`、`security`、`user`、`contract`、`review`、`agent`、`llm`、`rag`、`tool`、`approval`、`trace`、`infrastructure`。
- 配置 Spring Security、JWT、MyBatis Plus、Flyway、PostgreSQL、Redis、RabbitMQ、Actuator。
- 新增 RBAC 表、权限种子、角色权限绑定和默认开发账号 `admin/admin123`。
- 实现 `/auth/login` 与 `/auth/me`。
- 实现 Agent 状态机核心流转规则。
- 实现 LLM Gateway 抽象与 Mock 实现，覆盖 `chat`、`streamChat`、`embed`、`structuredOutput`、`toolCalling`。
- 补充 Agent 工具权限控制：新增 `review_tool_config`、`ToolPermissionGuard`，合同解析、条款抽取、规则检索、风险分析均接入工具权限校验。
- 补充知识库访问权限控制：新增 `KnowledgeAccessGuard`，知识库列表、文档列表和 RAG 检索按可见性、部门、角色和文档状态过滤。
- 补充异步审查执行器：合同审查任务改为 RabbitMQ 消息发布/消费驱动，消费端按 userId 重建用户上下文。

## 前端完成情况

- 建立 React + TypeScript + Vite + Ant Design 前端工程。

- 新增 API client 与 SSE EventSource 创建入口。

## 部署与本地环境

- 新增 `docker-compose.yml`，包含 PostgreSQL + pgvector、Redis、RabbitMQ，以及可选前端服务。
- 补充 RabbitMQ DirectExchange、队列、routing key 与 binding 声明，覆盖 `contract.review.queue`、`document.ingest.queue`、`tool.retry.queue`、`approval.event.queue`、`notification.queue`。
