# DB#07-审批表

## 任务目标

创建 approval_request 与 approval_history 表，支持 Human-in-the-loop 人工审批流程，记录审批对象、状态、意见、处理人和时间。

## 对应总览编号

```text
DB#07
```

## 涉及模块

```text
database: approval_request, approval_history
backend: approval, review, agent
```

## 本次实现

`V6__create_approval_request.sql` 新建两张表：

### approval_request

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | BIGSERIAL PK | 主键 |
| review_id | BIGINT FK | 关联审查任务 |
| approval_type | VARCHAR(32) | REVIEW / CLAUSE / REPORT |
| status | VARCHAR(32) | PENDING / APPROVED / REJECTED / REVISION_REQUESTED / ESCALATED / CANCELLED |
| approver_id | BIGINT FK | 审批人 |
| requested_by | BIGINT FK | 发起人 |
| comment | TEXT | 审批意见 |
| risk_summary | JSONB | 触发审批的风险摘要 |
| resolved_at | TIMESTAMPTZ | 审批完成时间 |

### approval_history

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | BIGSERIAL PK | 主键 |
| approval_request_id | BIGINT FK | 关联审批请求 |
| action | VARCHAR(32) | SUBMIT / APPROVE / REJECT / REQUEST_REVISION / ESCALATE / CANCEL / REASSIGN |
| operator_id | BIGINT FK | 操作人 |
| comment | TEXT | 操作备注 |
| metadata | JSONB | 扩展元数据 |

## 验收标准

- approval_request 支持审查任务级、单条款级、报告级三种审批
- approval_history 完整记录每次操作的时间线和操作人
- 与 AgentTaskStatus.WAITING_APPROVAL 状态对齐
- 索引覆盖常用查询字段
