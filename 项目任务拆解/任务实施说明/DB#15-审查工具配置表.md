# DB#15-审查工具配置表

## 任务目标

创建 review_tool 表，管理 Agent 审查工具的注册、权限、风险等级和启用状态。

## 对应总览编号

```text
DB#15
```

## 涉及模块

```text
database: review_tool
backend: tool, security
```

## 本次实现

`V10__create_review_tool.sql` 新建表：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | BIGSERIAL PK | 主键 |
| tool_name | VARCHAR(128) UNIQUE | 工具唯一名称 |
| display_name | VARCHAR(200) | 显示名称 |
| description | TEXT | 工具描述 |
| required_permission | VARCHAR(128) | 所需权限码 |
| risk_level | VARCHAR(20) | 工具风险等级 |
| approval_required | BOOLEAN | 调用是否需要审批 |
| enabled | BOOLEAN | 是否启用 |
| parameters_schema | JSONB | JSON Schema 参数定义 |
| department_scope | JSONB | 部门调用范围 |
| rate_limit | JSONB | 限流配置 |

内置三个初始工具：
- `contract_parser` — 合同文本解析器（LOW 风险）
- `clause_extractor` — 条款抽取器（LOW 风险）
- `risk_analyzer` — 条款风险分析器（MEDIUM 风险）

## 验收标准

- ToolPermissionGuard 可通过 required_permission 校验调用权限
- risk_level 和 approval_required 驱动高风险工具审批
- parameters_schema JSONB 支持参数验证
- enabled 支持工具热开关
