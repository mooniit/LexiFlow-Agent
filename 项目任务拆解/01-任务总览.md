## MVP 闭环任务清单

第一版必须优先形成以下闭环：

```text
登录与角色权限
 -> 合同上传
 -> 发起合同审查
 -> SSE 实时审查过程
 -> 合同文本解析
 -> 条款抽取
 -> RAG 检索合规规则
 -> 风险识别与修改建议
 -> 高风险触发审批
 -> 审批通过后生成最终报告
 -> Agent Trace 查看
 -> 合规知识库问答
```
