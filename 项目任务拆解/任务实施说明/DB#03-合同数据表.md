# DB#03-合同数据表

## 任务目标

完善 contract 表，保存合同文件元数据、合同状态、文件类型、上传人、金额、客户等信息。

## 对应总览编号

```text
DB#03
```

## 本次实现

- `V1__init_schema.sql` 已创建 `contract` 基础表，包含合同名称、类型、上传人、金额、客户、状态、文件路径等字段。
- `V3__enhance_contract_tables.sql` 补充索引：
  - `idx_contract_uploader_id` — 按上传人查询
  - `idx_contract_status` — 按状态筛选
  - `idx_contract_contract_type` — 按合同类型筛选
  - `idx_contract_customer_name` — 按客户名称搜索
  - `idx_contract_created_at` — 按创建时间排序

## 验收标准

- contract 表支持合同元数据存储
- 所有查询字段有对应索引
- Flyway 迁移无冲突
