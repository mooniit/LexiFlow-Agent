# DB#04-合同审查任务表

## 任务目标

完善 contract_review 表，保存审查任务、状态、失败原因、进度、关联合同。

## 对应总览编号

```text
DB#04
```

## 本次实现

- `V1__init_schema.sql` 已创建 `contract_review` 表，包含关联合同、审查状态、风险等级、失败原因、进度、结果摘要。
- `V3__enhance_contract_tables.sql` 补充索引：
  - `idx_contract_review_status` — 按状态筛选待处理任务
  - `idx_contract_review_overall_risk_level` — 按风险等级筛选
  - `idx_contract_review_created_by` — 按创建人查询
  - `idx_contract_review_created_at` — 按创建时间排序

## 验收标准

- contract_review 表支持审查任务完整生命周期
- 与 AgentTaskStatus 枚举（CREATED → COMPLETED/FAILED/CANCELLED）对齐
- 所有查询字段有对应索引
