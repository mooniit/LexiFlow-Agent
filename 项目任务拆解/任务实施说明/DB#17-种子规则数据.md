# DB#17-种子规则数据

## 任务目标

预置公司合同审查规范、模板和规则文档种子数据，保证本地启动即有可用的合规知识库。

## 对应总览编号

```text
DB#17
```

## 涉及模块

```text
database: knowledge_base, knowledge_document
backend: rag
```

## 本次实现

`V12__seed_knowledge.sql` 创建种子数据：

**知识库**：
- 公司合规规则库（PUBLIC / ACTIVE）

**规则文档**（6 份，均为 PUBLISHED 状态）：

| 文档标题 | 类型 | 说明 |
| --- | --- | --- |
| 公司合同审查规范 | POLICY | 核心审查规则与风险判定标准 |
| 销售合同标准模板 | TEMPLATE | 对外销售合同标准条款 |
| 采购合同标准模板 | TEMPLATE | 对外采购合同标准条款 |
| 数据保护条款指南 | GUIDELINE | 个人信息与数据保护合规指引 |
| 付款周期审批规则 | POLICY | 付款周期与金额的审批阈值 |
| 违约责任审查规则 | POLICY | 违约责任条款审查标准 |

## 验收标准

- 启动后知识库列表可查询到种子数据
- 种子文档状态为 PUBLISHED，可被 RAG 检索
- 幂等：重复执行不产生重复数据
