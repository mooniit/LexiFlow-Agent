# DB#10-工具调用日志表

## 任务目标

完善 tool_call_log 表，记录 Agent 每次审查工具调用的参数、结果、权限校验和耗时。

## 对应总览编号

```text
DB#10
```

## 涉及模块

```text
database: tool_call_log
backend: tool, trace
```

## 本次实现

- `V1__init_schema.sql` 已创建 `tool_call_log` 表，包含工具名、参数（JSONB）、结果（JSONB）、权限校验结果（JSONB）、耗时和成功/失败标记。
- `V7__enhance_agent_and_log_tables.sql` 补充索引：

| 新增索引 | 用途 |
| --- | --- |
| idx_tool_call_log_review_id | 按审查任务关联查询 |
| idx_tool_call_log_tool_name | 按工具名称统计调用次数和失败率 |
| idx_tool_call_log_success | 按成功/失败筛选 |
| idx_tool_call_log_created_at | 按时间排序 |

## 验收标准

- 支持按 review_id 追溯单次审查的所有工具调用
- permission_result JSONB 记录权限校验详情，支持审核追溯
- arguments / result JSONB 支持工具调用的输入输出审计
- 索引覆盖常用查询路径
