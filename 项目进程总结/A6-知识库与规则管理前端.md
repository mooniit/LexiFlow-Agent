# A6 知识库与规则管理前端

## 本阶段目标

完成合规知识库问答（FE#10）和规则库管理（FE#11）两个前端页面，对接后端 RAG 检索和知识库管理 API。

## 已完成任务

- FE#10 合规知识库问答页：提问 → RAG 检索 → 展示 chunk 内容和引用来源（文档 ID / Chunk ID / 相似度），支持问答历史记录。
- FE#11 规则库管理页：规则库列表、创建规则库、上传规则文档（TXT）、文档切片查看、批量导入。
