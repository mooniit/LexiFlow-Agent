# DB#05-合同解析与条款表

## 任务目标

创建 contract_clause 表，保存 LLM 从合同中抽取的结构化条款数据。

## 对应总览编号

```text
DB#05
```

## 涉及模块

```text
database: contract_clause
backend: contract, llm
```

## 本次实现

`V4__create_contract_clause.sql` 新建表，包含：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | BIGSERIAL PK | 主键 |
| contract_id | BIGINT FK | 关联合同 |
| clause_name | VARCHAR(200) | 条款名称 |
| clause_type | VARCHAR(64) | 条款类型枚举 |
| clause_text | TEXT | 条款原文 |
| structured_data | JSONB | LLM 结构化抽取结果 |
| sequence_order | INTEGER | 条款在合同中的出现顺序 |

支持的条款类型：parties / amount / payment_term / duration / liability / confidentiality / ip / data_protection / dispute_resolution / termination / indemnity_cap / auto_renewal

## 验收标准

- 表结构与 V1 审计字段规范一致
- contract_id 外键关联 contract 表
- JSONB 支持半结构化的条款字段存储
- 索引覆盖 contract_id 和 clause_type 查询
