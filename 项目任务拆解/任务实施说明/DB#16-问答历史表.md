# DB#16-问答历史表

## 任务目标

创建 qa_history 表，保存合规知识库问答的问题、回答、引用来源和用户反馈。

## 对应总览编号

```text
DB#16
```

## 涉及模块

```text
database: qa_history
backend: rag
```

## 本次实现

`V11__create_qa_history.sql` 新建表：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | BIGSERIAL PK | 主键 |
| user_id | BIGINT FK | 提问用户 |
| knowledge_base_id | BIGINT FK | 关联知识库 |
| question | TEXT | 用户问题 |
| answer | TEXT | 模型回答 |
| references_json | JSONB | 引用的文档和 chunk 来源 |
| retrieved_chunks | JSONB | RAG 检索到的相关 chunk |
| feedback | VARCHAR(16) | 用户反馈：HELPFUL/NOT_HELPFUL |

## 验收标准

- references_json 支持引用溯源展示
- feedback 支持回答质量统计
- 索引覆盖按用户、知识库和时间查询
