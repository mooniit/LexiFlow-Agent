# 目录使用示例

本目录用于集中维护 LexiFlow Agent 项目的任务拆解与协作约定。

建议使用方式：

- `01-任务总览.md`：记录所有待实现功能、数据库任务、部署任务、前端任务和测试任务。
- `任务实施说明/`：按任务编号创建实施说明文件，例如 `G#01-登录与权限模块.md`。
- 每个任务完成后，在 `01-任务总览.md` 中更新状态，并在对应实施说明中补充实现路径、关键文件、验证方式和遗留问题。

任务编号约定：

```text
G#   后端功能与业务能力
DB#  数据库、迁移、表结构、种子数据
DEP# 部署、环境、工程脚手架、中间件
FE#  前端页面与交互
TEST# 测试与质量保障
```

状态建议：

```text
TODO        待开始
DOING       进行中
BLOCKED     阻塞
DONE        已完成
DEFERRED    暂缓
```
